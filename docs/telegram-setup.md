# Подключение заявок к Telegram

Сайт отправляет все формы через серверный адрес `/api/leads`. Токен бота остаётся на сервере и не попадает в браузер посетителя.

## 1. Создать группу и бота

1. Создайте в Telegram закрытую группу, например «Заявки — Перекрёсток».
2. Откройте официального бота `@BotFather` и выполните команду`.
3. Сохраните выданный токен в надёжном месте. Не присылайте его в чат, не вставляйте в код и не загружайте на GitHub.
4. Добавьте нового бота в группу и разрешите ему отправлять сообщения.
5. Отправьте в группе команду `/start@имя_вашего_бота`.

## 2. Узнать ID группы

В PowerShell временно задайте токен и запросите обновления:

```powershell
$env:TELEGRAM_BOT_TOKEN = Read-Host "Вставьте токен бота"
Invoke-RestMethod "https://api.telegram.org/bot$env:TELEGRAM_BOT_TOKEN/getUpdates" | ConvertTo-Json -Depth 8
```

В ответе найдите `message.chat.id`. ID закрытой группы обычно начинается со знака минус, например `-1001234567890`. После этого закройте окно PowerShell.

## 3. Настроить локальный запуск

Создайте файл `.env.local` рядом с `package.json`:

```env
TELEGRAM_BOT_TOKEN=токен_от_BotFather
TELEGRAM_CHAT_ID=-1001234567890
```

Файл `.env.local` уже исключён из Git и не будет загружен на GitHub.

## 4. Настроить Vercel

1. Импортируйте GitHub-репозиторий проекта в Vercel.
2. В `Settings → Environment Variables` добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`.
3. Выберите окружения Production, Preview и Development.
4. Выполните новый Deploy.

Для отправки заявок нужен серверный хостинг. Статический GitHub Pages для этой версии больше не подходит; GitHub остаётся хранилищем кода, а сайт запускается через Vercel.

## 5. Проверка

Откройте опубликованный сайт, заполните форму тестовыми данными, подтвердите согласие и отправьте её. В закрытой группе должно появиться сообщение с источником заявки, контактами и выбранной программой. Затем отдельно проверьте контактную форму и финал квиза.

Если Telegram временно недоступен, сайт покажет понятную ошибку и сохранит введённые поля, чтобы посетитель мог повторить отправку или позвонить.
